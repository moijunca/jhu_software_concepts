This project was developed as part of Module 1 for Software Concepts.
It demonstrates a multi-page web application built with Python and Flask.

Features:
- Flask application with multiple routes
- Jinja2 templates
- Static assets (CSS and images)
- Projects and Contact pages
- Version control using Git and GitHub

How to Run the Application:

1. Navigate to the module_1 directory:
   cd module_1

2. Create a virtual environment:
   python3 -m venv .venv

3. Activate the virtual environment:
   source .venv/bin/activate

4. Install dependencies:
   pip install -r requirements.txt

5. Run the Flask application:
   python run.py

6. Open a browser and visit:
   http://127.0.0.1:8080
